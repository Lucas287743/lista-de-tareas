# Lista de Tareas
Una aplicación web para gestionar tareas con funciones de arrastrar y soltar y edición en línea.

# Funcionalidades
- Edición de celdas mediante un modal.
- Reordenamiento de tareas mediante arrastre.
- Persistencia de datos en `localStorage`.

# Tecnologías
- HTML5, CSS3, JavaScript (Vanilla).
- FontAwesome para los iconos.
